# Structure

The file structure of the `core` is reflected in `mods`.

```
core/
├── assets/
│   ├── fonts/
│   │   ├── FiraSans-Bold.ttf
│   ├── images/
│   │   ├── background.png
│   ├── textures/
│   │   ├── texture_map.png
├── data/
│   ├── character.toml
│   ├── classes.toml
│   ├── effects.toml
│   ├── items.toml
│   ├── locations.toml
│   ├── magic.toml
│   ├── missions.toml
│   ├── mobs.toml
│   ├── races.toml
├── checksum
├── mods.toml
solid
```
